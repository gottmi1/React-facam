
각 특징

1. onsenui

모바일 대응이 좋음
하이브리드 웹앱을 만들 때 자주 사용됨
React에 불친절함.

2. Ant Design

React 베이스
중국에서 많이 씀
컴포넌트 제공 + 디자인 철학
레이아웃 : 배치가 간편
다양한 컴포넌트 : 복붙만으로 화면구성이 가능

3. React Semantic UI

웹 접근성에 대한 UI(시각장애인 등)
원래 제이쿼리 기반이지만 리액트용으로 제공
as props로 시멘틱한 태그를 사용 가능
커멘트 / 피드 - 다 만들어서 제공
웹 서비스 구성에 필요한 거의 모든 컴포넌트를 제공
여러 컴포넌트의 조합

4. 리액트 부트스트랩

xs/ md/ lg 등의 크기가 정해져있어 반응형웹에 용이
Sass @import해서 쓸수도 있음
컬러시스템 : 통일성 있음
유틸리티(트랜지션, Ratio등)으로 확장성이 높다

5. Material UI

머테리얼 디자인 -> 구글이 선도한 디자인 시스템
faq로 실무과정에서 마주할 이슈들이 잘 정리되어있음
레이아웃 / 테마 등이 쳬계화됨.

6. Tailwind CSS

컴포넌트 자체를 제공하지는 않음(유료)
class로 스타일링 요소를 제공하긴 함
Tiny : 빌드할 때 사용하는 class만 임포트할 수 있다


---- 여기까진 UI라이브러리 이 아래론 애니메이션 라이브러리

7. React spring

물리엔진을 사용한 좀 더 나은 애니메이션
hooks 제공
animated -> 리액트 밖에서 관장 가능

8. Framer Motion

애니메이션 기능이 들어간 컴포넌트를 사용할 수 있다
제스쳐, 모션벨류등 편의기능 제공


새로운 라이브러리를 적용하는 경험

필요에 따라 찾고 -> 공식문서로 파악 -> 공식문서를 따라 설치 -> 공식문서 예제를 따라해보고 api문서를 보고 적용해 본다

어느 라이브러리든 비슷하다 쫄 것 없음.
